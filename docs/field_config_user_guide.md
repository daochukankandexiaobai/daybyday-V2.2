# 字段与报表配置使用说明

## 入口

1. 使用管理员菜单登录。
2. 登录成功后，顶部页签会出现“字段与报表配置”。
3. 普通经理模式不会显示该页签，也不能修改全局字段配置。

## 新增字段

在“字段管理”页点击“新增字段”，填写：

- 字段编码：只能使用英文小写、数字、下划线，例如 `new_customer_count`
- 显示名称：页面和导出中显示的中文名
- 字段类型：常用为 `int`、`amount`、`money`、`decimal`、`text`
- 字段分类：日报录入字段请选择 `raw_daily`
- 所属分组：例如 `process_behavior`
- 默认值：数值字段必须填写数字
- 统计方式：数量和金额通常使用 `sum`

勾选“参与页面 / 导出”后保存。新字段会进入对应页面配置。

## 停用字段

选中字段后点击“停用字段”。

停用只会设置 `enabled = 0`：

- 不删除历史日报数据
- 不删除动态指标值
- 不再显示在已配置页面中

系统字段不会被物理删除。

## 配置数据录入页

进入“数据录入配置”：

1. 勾选字段是否显示。
2. 使用“置顶 / 上移 / 下移 / 置底”调整顺序。
3. 点击“保存配置”。
4. 如需恢复稳定版本默认顺序，点击“恢复默认”。

## 配置今日展示 / 查询汇总

进入“今日展示 / 查询汇总配置”：

1. 选择配置页面。
2. 勾选字段显示。
3. 修改分组或顺序。
4. 点击“保存配置”。

查询汇总仍保持“一人一行聚合”，字段停用不会删除历史数据。

## 配置 PNG 导出

进入“PNG 导出配置”：

1. 选择 PNG 默认模板。
2. 编辑 JSON 中的 `sections`。
3. 每张图的 `field_keys` 建议不超过 14 个。
4. 点击“保存模板”。

如果模板损坏或字段不存在，系统会跳过异常字段或回退默认模板。

## 配置备份与恢复

页面顶部提供：

- 导出配置：备份字段、页面可见性、模板配置到 JSON
- 导入配置：从 JSON 恢复配置
- 恢复全部默认：停用自定义字段，恢复系统默认字段顺序和模板

建议在批量修改字段前先导出配置备份。
